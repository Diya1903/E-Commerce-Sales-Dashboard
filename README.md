
#  E-Commerce Sales & Customer Analytics Dashboard (Power BI)

##  Project Overview

This project presents an interactive **E-Commerce Sales Dashboard** built using Power BI. The dashboard provides insights into sales performance, customer behavior, and regional trends to support data-driven decision-making.

---

##  Objectives

* Analyze overall sales and profit
* Identify top-performing products and categories
* Understand customer purchasing behavior
* Track sales trends over time
* Provide interactive filtering for better insights

---

##  Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Power Query (Data Cleaning & Transformation)

---

##  Dataset

* Source: Kaggle Superstore Dataset
* Link: https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

---

## Dashboard Features

### KPIs

* Total Sales
* Total Profit
* Total Orders
* Average Order Value

### Visualizations

* Sales Trend (Time Series)
* Sales by Category
* Top Customers Table
* Customer Segment Analysis

###  Filters (Slicers)

* Date
* Region
* Category

---

##  Data Cleaning Steps

* Removed null values and errors
* Converted data types (Date, Numeric)
* Removed duplicate records
* Transformed data using Power Query

---

##  Key Insights

* Sales show an increasing trend over time
* Certain categories contribute higher revenue
* Top customers generate a significant portion of sales
* Regional performance varies significantly

---

##  Project Outcome

This dashboard helps stakeholders:

* Monitor business performance
* Identify growth opportunities
* Make informed decisions using data
